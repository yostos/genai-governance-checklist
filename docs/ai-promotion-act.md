# AI推進法の調査と本チェックリストへの影響分析

## 1. 法律の基本情報

| 項目 | 内容 |
| ---- | ---- |
| 正式名称 | 人工知能関連技術の研究開発及び活用の推進に関する法律 |
| 通称 | AI法、AI推進法 |
| 法律番号 | 令和7年法律第53号 |
| 成立 | 2025年5月28日 |
| 公布・一部施行 | 2025年6月4日 |
| 全面施行 | 2025年9月1日 |
| 性質 | 基本法的性質の理念型法律（直接的な罰則規定なし） |

日本初のAIに特化した法律である。
EU AI Actのようにリスクレベルに応じた義務を課す規制法ではなく、
基本理念と政府の施策方針を定める基本法である。
事業者への直接的な罰則はないが、
政府による指導・助言の仕組みを通じて実効性を確保する。

## 2. 法律の構成（全4章28条）

| 章 | 条文 | 内容 |
| -- | ---- | ---- |
| 第1章 総則 | 第1条〜第8条 | 目的、定義、基本理念、各主体の責務 |
| 第2章 基本的施策 | 第9条〜第17条 | 研究開発推進、人材育成、適正利用等 |
| 第3章 人工知能基本計画 | 第18条 | 政府によるAI基本計画の策定義務 |
| 第4章 人工知能戦略本部 | 第19条〜第28条 | AI政策の司令塔設置 |

## 3. チェックリストに関連する主要条文

### 3.1 第2条（定義）

「人工知能関連技術」の法的定義を置いている。
「人工的な方法により人間の認知、推論及び判断に係る
知的な能力を代替する機能を実現するために必要な技術
並びに入力された情報を当該技術を利用して処理し、
その結果を出力する機能を実現するための
情報処理システムに関する技術」と定義される。

### 3.2 第3条（基本理念）

基本理念は5項で構成される。
チェックリストの方向性に直結する条文である。

| 項 | 要旨 | チェックリストとの関連 |
| -- | ---- | ---------------------- |
| 第1項 | 科学技術・イノベーション基本法およびデジタル社会形成基本法の基本理念を踏まえる | （上位法との整合性） |
| 第2項 | AI技術は経済社会・安全保障上重要であり、研究開発力保持・国際競争力向上のために推進する | 2.1節（利用目的・文脈の明確化） |
| 第3項 | 基礎研究から実用化まで総合的・計画的に推進する | 1.2節（ポリシー・規程） |
| 第4項 | 不正目的・不適切方法によるAI利用は犯罪利用・個人情報漏えい・著作権侵害等のリスクがあり、過程の透明性確保等が必要 | ch03（入力データ管理）、ch04（出力管理）、5.4節（透明性） |
| 第5項 | 国際的協調のもとで推進し、日本が主導的役割を果たす | references.md（国際基準の参照） |

**第4項が最も重要**である。
AI利用に伴う具体的リスクとして
「犯罪への利用、個人情報の漏えい、著作権の侵害
その他国民生活の平穏及び国民の権利利益が
害される事態」を明示し、
「過程の透明性の確保その他の必要な施策」を
求めている。

### 3.3 第7条（活用事業者の責務）

「活用事業者」はAI事業者ガイドラインにおける
「AI開発者」「AI提供者」「AI利用者」の
すべてを包含する広い概念である。

事業者には2つの責務がある。

1. **努力義務**: 自ら積極的にAI技術を活用し、
   事業活動の効率化・高度化・新産業創出に努める
2. **協力義務**: 国・地方公共団体が実施する
   施策に協力しなければならない

直接的な罰則はないが、
第16条に基づく調査・指導の対象となる。

### 3.4 第13条（適正性の確保のための指針の整備等）

国が「国際的な規範の趣旨に即した指針の整備
その他の必要な施策を講ずる」と定める。

広島AIプロセス国際指針を踏まえ、
以下が指針に含まれる見込みである。

- 不適切な出力を減らすための開発者側の工夫
- AIの概要・リスク・学習データ情報の開示・共有
- 人間中心のAI原則（安全性・人権等）の確保

### 3.5 第16条（調査研究等・指導助言）

国は以下を実施する。

- AI関連技術の国内外動向の情報収集
- 不正目的・不適切方法によるAI利用で
  国民の権利利益侵害が生じた事案の分析
- 事業者への**指導・助言・情報提供**
  その他の必要な措置

罰則はないものの、
指導に応じない事業者に対しては
事業者名の公表等の措置を検討されうる。

## 4. 人工知能基本計画との関係

AI推進法第18条に基づき、
令和7年12月23日に「人工知能基本計画」が閣議決定された。
副題は「『信頼できるAI』による『日本再起』」である。

基本方針として、AIを「使う」「創る」
「信頼性を高める」「AIと協働する」の
4つの施策を掲げている。

「イノベーション促進とリスク対応の両立」
「アジャイル（柔軟かつ迅速）な対応」
「内外一体での政策推進」を原則としている。

本チェックリストが前提とする
「リスク管理とAI活用の両立」の方向性は、
AI基本計画の方針と整合している。

## 5. チェックリスト既存項目との対応分析

AI推進法は基本法であり、
事業者に対して具体的な行為義務を直接課す条文はほぼない。
したがって、チェックリストに新規項目を追加する
必要性は低い。

ただし、以下の既存項目で
AI推進法への言及を追加することが適切である。

### 5.1 対応が必要な項目

**1.2.D**（関連する法令・規制要件への言及）

現在の定義例は個人情報保護法・著作権法・
不正競争防止法を列挙している。
AI推進法を追加することで、
AI固有の法律が存在することを明示できる。

> 定義例への追記案:
> 「人工知能関連技術の研究開発及び活用の推進に関する法律
> （AI推進法）」を関連法令に追加

**references.md**

参照ガイドライン一覧の
「日本（政府・公的機関）」セクションに
AI推進法と人工知能基本計画を追加する。

### 5.2 対応不要と判断した理由

以下の理由により、
新規チェック項目の追加は不要と判断する。

1. **理念法であること**: AI推進法は
   事業者に具体的な行為義務を課していない。
   第7条の責務も努力義務が中心である
2. **既存項目でカバー済み**: AI推進法第3条第4項が
   示すリスク（個人情報漏えい・著作権侵害・透明性確保）は、
   本チェックリストのch03・ch04・5.4節で
   すでに手当てされている
3. **ガイドラインの位置づけ**: AI推進法第13条に基づく
   指針は今後策定される。
   現時点ではAI事業者ガイドライン（METI）が
   実質的な基準であり、
   本チェックリストはすでにこれを参照している
4. **対象組織との適合性**: 本チェックリストの対象は
   SaaS型生成AIの利用者（小規模組織）であり、
   AI推進法が主に想定するAI開発者・提供者とは
   位置づけが異なる
